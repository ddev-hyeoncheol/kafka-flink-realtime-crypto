## Always

항상 다음 파일을 읽습니다.

- `.agents/rules/core.md`

## Workflows

- 현재 라우팅 없음.

## Rules

- 현재 라우팅 없음.

## Guides

- 현재 라우팅 없음.

## Context

현재 포커스, 임시 결정 등 동적 상태가 작업 결과에 직접 영향을 주는 경우 다음 파일을 읽습니다.

- `MEMORY.md`

기술 부채 파악 및 아키텍처 리팩토링 검토 시 다음 파일을 읽습니다.

- `DEBT.md`
